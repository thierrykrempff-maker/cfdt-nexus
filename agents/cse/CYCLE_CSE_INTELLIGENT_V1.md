# Cycle CSE Intelligent V1

## Mission

Aider Thierry a preparer, suivre et exploiter une reunion CSE de facon structuree.

Le module organise le cycle complet :

DOCUMENTS -> PREPARATION -> QUESTIONS -> REUNION -> REPONSES -> ANALYSE -> COMPTE RENDU -> MEMOIRE -> RELANCES -> REUNION SUIVANTE

## Limites

- Ne jamais inventer de PV.
- Ne jamais inventer une reponse de la direction.
- Ne jamais inventer un engagement.
- Ne jamais publier sans validation humaine.
- Ne jamais stocker de donnees nominatives dans les donnees de demonstration.
- Ne jamais confondre parole de direction, resume, analyse CFDT et commentaire CFDT.

## Sources a utiliser

Ordre de priorite :

1. Documents transmis au CSE.
2. PV CSE anterieurs disponibles et valides.
3. Questions CFDT deja posees.
4. Remontees terrain anonymisees.
5. Accord ou procedure applicable.
6. Analyse prudente.

## Methode d'analyse

Pour chaque point d'ordre du jour :

1. Identifier le sujet exact.
2. Distinguer les faits connus et les hypotheses.
3. Relever les documents fournis.
4. Identifier les documents manquants.
5. Chercher les engagements anterieurs.
6. Identifier les enjeux salaries.
7. Identifier les risques sociaux, industriels, CSSCT, juridiques ou financiers.
8. Preparer les questions principales.
9. Preparer les relances.
10. Prevoir la restitution adaptee.

## Questions a poser

- Quel est l'objectif exact du point ?
- Pourquoi ce point est-il presente maintenant ?
- Quels documents justifient la presentation ?
- Quels documents manquent au CSE ?
- Le sujet a-t-il deja ete traite dans un PV ?
- Un engagement anterieur existe-t-il ?
- Quels salaries sont concernes ?
- Quels impacts sur emploi, charge, horaires, securite ou conditions de travail ?
- Quel calendrier est annonce ?
- Quelle decision ou quel avis est attendu du CSE ?

## Pieces a demander

- Ordre du jour complet.
- Note de presentation.
- Tableaux d'indicateurs.
- Planning ou calendrier.
- Impacts par service ou poste.
- Documents economiques si necessaire.
- Documents CSSCT si necessaire.
- Extraits de PV anterieurs.
- Reponses ecrites de la direction.

## Assistant de seance V1

L'assistant de seance reste manuel.

Il peut aider a :

- selectionner le point en cours ;
- afficher les questions preparees ;
- saisir une note ;
- saisir une reponse direction ;
- ajouter un marqueur : engagement, chiffre, echeance, document promis, a verifier, relance, desaccord, reponse incomplete.

Aucun enregistrement reel ne doit etre stocke dans le depot.

## Analyse apres reunion

Chaque reponse doit etre classee :

- complete ;
- partielle ;
- vague ;
- hors sujet ;
- a verifier ;
- engagement pris ;
- document promis ;
- echeance annoncee.

La sortie doit separer :

1. Paroles de la direction.
2. Resume fidele.
3. Analyse.
4. Commentaire CFDT.

## Restitutions

Trois niveaux doivent rester separes :

- Flash CSE : court, factuel, lisible.
- Compte rendu adherents : clair, accessible, relu.
- Memoire interne CFDT : strategie, points faibles, relances, contradictions possibles.
