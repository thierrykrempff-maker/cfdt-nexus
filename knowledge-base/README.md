# Base documentaire

La base documentaire contient les sources utilisées par CFDT Nexus.

## Classement

- `public/` : documents publics validés ;
- `internal/` : documents internes non diffusables sans validation ;
- `conventions/` : convention collective et textes conventionnels ;
- `accords/` : accords INEOS disponibles ;
- `jurisprudence/` : décisions et analyses jurisprudentielles ;
- `sources/` : sources officielles, liens et références.

## Sources externes

Le dossier `sources/` contient maintenant le registre V1 des sources juridiques, sociales, CFDT, branche Chimie et veille spécialisée.

Fichiers principaux :

- `sources.registry.json` : registre structuré des sources ;
- `source-registry.schema.json` : modèle de données d'une source ;
- `SOURCE_TRUST_HIERARCHY.md` : hiérarchie de confiance A / B / C / D.

Aucune source secondaire ou sociale ne doit être utilisée seule pour produire une conclusion juridique.

## Règles

- Ne pas ajouter de donnée personnelle réelle sans nécessité et validation.
- Ne pas exposer les documents internes dans les usages publics.
- Indiquer la date et l'origine des documents importants.
- Prévoir une vérification humaine avant toute exploitation sensible.
