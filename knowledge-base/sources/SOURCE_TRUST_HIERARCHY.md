# Hiérarchie de confiance des sources V1

CFDT Nexus utilise une hiérarchie stricte pour éviter de transformer une veille en affirmation juridique non vérifiée.

## Niveaux

### A - Source primaire opposable

Sources à privilégier pour établir une règle, un texte applicable ou une référence juridique :

- Légifrance ;
- Code du travail numérique ;
- Cour de cassation ;
- Conseil d'État ;
- Conseil constitutionnel ;
- BOSS ;
- URSSAF ;
- Service-Public.fr ;
- Ministère du Travail ;
- textes conventionnels publiés officiellement.

### B - Source institutionnelle ou experte

Sources fiables pour comprendre, prévenir ou contextualiser :

- INRS ;
- ANACT ;
- Assurance Maladie - Risques professionnels ;
- DARES ;
- INSEE ;
- France Chimie ;
- CFDT ;
- FCE-CFDT.

### C - Veille spécialisée

Sources utiles pour détecter des sujets ou préparer une analyse, mais jamais suffisantes seules :

- Actuel RH ;
- Éditions Tissot ;
- Liaisons Sociales ;
- Dalloz Actualité ;
- Village de la Justice ;
- revues professionnelles spécialisées.

### D - Signal faible

Réseaux sociaux, newsletters, communiqués, alertes ou reprises d'information.

Un signal faible peut déclencher une vérification, mais ne doit jamais fonder une réponse ou une publication sans source A ou B.

## Règles absolues

- Ne jamais inventer une règle, une jurisprudence, un accord ou une source.
- Ne jamais publier automatiquement une veille.
- Toujours distinguer fait, source, analyse, niveau de confiance et action proposée.
- Toujours vérifier la date de publication et la date d'entrée en vigueur.
- Toujours rechercher la source primaire lorsqu'une information provient d'une source secondaire.
- Toujours signaler les contenus payants, partiels ou non vérifiés automatiquement.
