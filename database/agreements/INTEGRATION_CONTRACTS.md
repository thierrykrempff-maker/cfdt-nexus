# Contrats d'intégration - Bible Accords Sarralbe V1

## Principe

La Bible Accords Sarralbe V1 fournit une recherche documentaire privée. Elle ne produit pas une analyse juridique automatique.

Les sorties doivent toujours distinguer :

- recherche documentaire ;
- comparaison textuelle ;
- analyse juridique ;
- décision humaine.

## Document Intelligence Center

Le DIC pourra appeler la Bible Accords pour :

- rechercher dans les accords locaux ;
- sélectionner des passages pertinents ;
- afficher une citation ;
- repérer un texte potentiellement lié ;
- comparer un nouveau projet avec des textes antérieurs.

Contrat de sortie attendu :

```text
QUESTION
REPONSE PROVISOIRE
NIVEAU DE CONFIANCE
SOURCES UTILISEES
POINTS A VERIFIER
CONFLITS EVENTUELS
DOCUMENT PLUS RECENT POTENTIELLEMENT LIE
ACTION HUMAINE RECOMMANDEE
```

## Cycle CSE

Le Cycle CSE pourra demander :

- rechercher les accords liés au temps de travail ;
- rechercher les textes liés à la rémunération ;
- rechercher le règlement intérieur ou les textes collectifs liés à une procédure disciplinaire ;
- identifier les documents à demander à la direction.

La provenance doit rester claire :

- corpus privé Sarralbe ;
- convention collective ;
- loi ;
- jurisprudence ;
- source institutionnelle ;
- analyse CFDT.

## Agents spécialisés

La Bible prépare le routage documentaire vers :

- Expert Accords Sarralbe ;
- Expert Convention Chimie ;
- Expert CSE ;
- Expert CSSCT ;
- Défenseur Syndical ;
- Expert Rémunération ;
- Expert Temps de Travail ;
- Expert Analyse Financière.

Un même passage peut servir plusieurs agents.

## Bouton "Que me manque-t-il ?"

La sortie doit rester une aide à la décision :

- correction humaine possible ;
- option non applicable ;
- note terrain possible ;
- validation humaine d'un document comme suffisant.
