# Roadmap

## Phase 1 - Socle projet

- Créer l'architecture documentaire.
- Définir les rôles des agents spécialisés.
- Formaliser les règles de validation humaine.
- Identifier les premiers workflows utiles.

## Phase 2 - Agents spécialisés

- Décrire les responsabilités de chaque agent.
- Préparer les prompts de base.
- Structurer les sources et documents de référence.
- Définir les limites d'utilisation.

## Phase 3 - Automatisations

- Documenter les scénarios n8n.
- Prévoir les contrôles avant diffusion.
- Mettre en place une logique de suivi et d'amélioration.

## Phase 4 - Interface

- Préparer les contenus du site ou tableau de bord.
- Organiser les accès par usage.
- Tester les parcours avec des cas simples.
