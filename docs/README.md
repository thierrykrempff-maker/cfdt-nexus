# Documentation

Le dossier `docs/` regroupe la documentation durable du projet CFDT Nexus.

## Zones

- `vision/` : objectifs, principes et trajectoire produit ;
- `architecture/` : architecture globale et technique ;
- `decisions/` : décisions d'architecture ;
- `security/` : sécurité, confidentialité et risques ;
- `data-governance/` : règles de classement et gestion documentaire ;
- `workflows/` : procédures fonctionnelles ;
- `operations/` : exploitation, maintenance et supervision.
