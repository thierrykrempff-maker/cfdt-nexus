# Methodologies CFDT Nexus

Ce dossier regroupe les methodes de travail versionnees utilisees par CFDT Nexus.

Les methodologies ne sont pas des sources juridiques ou economiques. Elles decrivent une facon de travailler, les controles a effectuer, les limites a respecter et les restitutions attendues.

## Methodes disponibles

- `CYCLE_CSE_INTELLIGENT_V1.md` : preparation, suivi et exploitation d'une reunion CSE.
- `ANALYSE_FINANCIERE_SARRALBE_V1.md` : lecture prudente des donnees economiques Sarralbe avec donnees de demonstration fictives.

## Regles

- Ne jamais inventer de PV, de reponse direction, de chiffre ou d'engagement.
- Toujours separer faits, hypotheses, analyse et commentaire CFDT.
- Toujours maintenir une validation humaine avant publication.
- Toujours garder les donnees reelles hors depot tant qu'un stockage securise n'existe pas.
