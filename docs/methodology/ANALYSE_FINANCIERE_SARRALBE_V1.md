# METHODE CFDT NEXUS
## Analyse financiere Sarralbe

Version : 1.0

## Objectif

Cette methode aide un elu CSE a lire les donnees economiques utiles pour Sarralbe, sans tirer de conclusion strategique non demontree.

Le module est centre sur Sarralbe. Les autres sites ne servent de comparaison que si cela aide a comprendre un arbitrage, une allocation de volumes, un investissement ou une consequence possible pour Sarralbe.

## Contexte economique

Sarralbe fonctionne selon un modele de toll manufacturing / faconnage.

Le site n'est pas proprietaire des matieres premieres entrantes ni des produits finis sortants. Le modele local est de type cost-plus avec une marge contractuelle de 3 %.

La lecture CSE doit donc chercher la chaine :

VOLUMES -> PRODUCTION -> VENTES GROUPE -> MARGE BRUTE -> EBITDA RC -> ECART AU BUDGET -> EFFETS HC / STOCKS EVENTUELS -> DECISIONS GROUPE POSSIBLES -> CONSEQUENCES POSSIBLES SUR LA CHARGE DE SARRALBE -> CONSEQUENCES SOCIALES ET INDUSTRIELLES

Toute conclusion non directement demontree doit etre formulee comme :

HYPOTHESE A VERIFIER

## Activites a lire separement

- Sarralbe HDPE ;
- Sarralbe PP.

Pour chaque activite :

- Production actual / budget / ecart / evolution ;
- Sales actual / budget / ecart / relation production-ventes ;
- Gross Margin actual / budget / ecart / evolution ;
- RC EBITDA actual / budget / ecart / tendance ;
- HC EBITDA actual / budget / ecart / difference avec RC.

## Regles d'analyse

### Volumes

Lorsque les volumes sont durablement sous budget, analyser le risque sur la charge industrielle et l'absorption des frais fixes.

Questions prioritaires :

- cause commerciale, industrielle ou arbitrage de volumes ;
- charge prevue a 3 mois, 6 mois et 12 mois ;
- maintien ou revision du budget ;
- forecast actualise.

### RC et HC

Ne pas confondre automatiquement EBITDA RC et EBITDA HC.

L'analyse doit :

- examiner la performance RC ;
- identifier l'ecart HC ;
- rechercher les explications disponibles ;
- eviter toute conclusion non demontree.

### Marge et volumes

Une marge superieure au budget ne compense pas necessairement un manque de volume.

Si marge superieure au budget + volume inferieur au budget + RC EBITDA inferieur au budget :

- isoler effet volume ;
- isoler effet marge ;
- isoler effet couts ;
- verifier disponibilite et autres facteurs.

### Investissements

Pour les projets industriels, ne pas analyser uniquement le retour sur la marge locale.

Relier les investissements a :

- rendement matiere ;
- energie ;
- disponibilite ;
- fiabilite ;
- maintenance ;
- capacite ;
- qualite ;
- reduction de couts ;
- performance RC ;
- perennite de l'outil ;
- charge, emploi, conditions de travail et securite.

## Double lecture obligatoire

Chaque analyse importante doit contenir :

- Vision Groupe / Principal ;
- Consequences possibles pour Sarralbe ;
- Questions CSE prioritaires ;
- Points a surveiller.

L'interface doit rester centree sur Sarralbe.

## Donnees de demonstration

Les donnees de demonstration doivent etre fictives.

Ne jamais recopier un chiffre reel communique au Product Owner dans les fixtures.
