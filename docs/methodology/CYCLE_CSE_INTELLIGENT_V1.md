# METHODE CFDT NEXUS
## Cycle CSE intelligent

Version : 1.0

## Objectif

Cette methode structure le travail d'un elu CSE avant, pendant et apres une reunion.

Elle ne remplace pas l'analyse humaine. Elle organise les informations, aide a preparer les questions, trace les engagements et separe clairement les faits, les hypotheses, les reponses de la direction et les commentaires CFDT.

## Cycle central

DOCUMENTS -> PREPARATION -> QUESTIONS -> REUNION -> REPONSES -> ANALYSE -> COMPTE RENDU -> MEMOIRE -> RELANCES -> REUNION SUIVANTE

## Avant la reunion

Pour chaque point inscrit par la direction, preparer une fiche avec :

- sujet exact ;
- intention probable de presentation ;
- contexte ;
- documents fournis ;
- documents manquants ;
- sujets similaires dans les PV ;
- engagements anterieurs ;
- echeances ;
- enjeux salaries ;
- risques ;
- chiffres a verifier ;
- questions et relances.

Les indicateurs a afficher sont :

- analyse documentaire necessaire ;
- recherche PV necessaire ;
- analyse juridique necessaire ;
- analyse financiere necessaire ;
- analyse CSSCT necessaire ;
- avis motive a preparer ;
- consultation des salaries a envisager.

## Pendant la reunion

L'assistant de seance V1 doit rester manuel.

Il doit permettre :

- selection du point en cours ;
- consultation des questions preparees ;
- prise de notes ;
- saisie manuelle d'une reponse de la direction ;
- marqueurs rapides : engagement, chiffre, echeance, document promis, a verifier, relance, desaccord, reponse incomplete.

Chaque marqueur est rattache a la reunion, au point, a la question si elle existe et au moment de saisie.

Aucun vrai enregistrement ne doit etre stocke dans le depot.

## Apres la reunion

L'analyse doit distinguer :

1. paroles de la direction ;
2. resume fidele ;
3. analyse ;
4. commentaire CFDT.

Les reponses peuvent etre classees :

- complete ;
- partielle ;
- vague ;
- hors sujet ;
- a verifier ;
- engagement pris ;
- document promis ;
- echeance annoncee.

## Restitutions

Trois niveaux doivent rester separes :

- Flash CSE : lecture courte pour retenir l'essentiel ;
- Compte rendu adherents : information claire, factuelle et accessible ;
- Memoire interne CFDT : strategie, points faibles, relances, contradictions possibles et actions futures.

Validation humaine obligatoire avant toute publication.

## Limites

Ne jamais inventer de PV, de jurisprudence, d'engagement ou de reponse de direction.

Les donnees de demonstration doivent etre fictives et ne doivent contenir aucune donnee nominative.
