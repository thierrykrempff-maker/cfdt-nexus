# Tests

Ce dossier contiendra les tests de CFDT Nexus.

## Zones

- `agents/` : tests de comportement des agents ;
- `prompts/` : tests et cas de prompts ;
- `workflows/` : tests de workflows ;
- `site/` : tests du site ou des interfaces.

## Objectifs

- Vérifier les règles de sécurité.
- Vérifier que les agents n'inventent pas de règles.
- Vérifier que les workflows ne publient pas sans validation.
- Vérifier accessibilité, responsive et SEO du site.
