# knowledge-private.example

Ce dossier est un exemple de structure pour une future base documentaire privée.

Il ne doit contenir aucun vrai document.

## Important

Ne jamais ajouter ici :

- accord INEOS réel ;
- PV CSE réel ;
- BDESE ;
- règlement intérieur réel non validé pour versionnage ;
- dossier salarié ;
- document nominatif ;
- document médical ;
- bulletin de paie ;
- fichier confidentiel.

## Usage prévu

Les vrais documents doivent rester hors GitHub :

- sur le PC de Thierry ;
- dans un cloud privé maîtrisé ;
- dans un espace chiffré ou contrôlé ;
- dans un futur backend sécurisé.

Ce dossier sert uniquement à montrer l'organisation cible.

## Structure exemple

```text
knowledge-private.example/
  convention-chimie/
  accords-entreprise/
  reglement-interieur/
  pv-cse/
  jurisprudence/
  modeles/
  veille/
```

Chaque dossier est conservé par un fichier `.gitkeep` pour éviter d'ajouter de vrais documents.
