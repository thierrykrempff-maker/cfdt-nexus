# Prompts

Ce dossier contient les prompts réutilisables de CFDT Nexus.

## Zones

- `codex/` : prompts destinés aux modifications de dépôt, site ou automatisations ;
- `chatgpt/` : prompts destinés aux échanges utilisateur ;
- `system/` : prompts système ou prompts centraux.

## Règles

- Versionner les prompts importants.
- Éviter les consignes contradictoires.
- Documenter les limites.
- Prévoir des tests pour les prompts sensibles.
